# Security Policy

## Supported Versions

As aether-arena doesn't provide an offical release yet, please use the latest version for the security updates.
Current we have two branches to maintain: 
* main branch for aether-arena 2.x
* main-1.x branch for aether-arena 1.x 

## Reporting a Vulnerability

Please go to https://github.com/bytedance/aether-arena/security to report the vulnerability you find.
